## Post
```shell
filename=/Users/mac/GitHub/zhousongjie-next/content/posts/untitled
echo --- > $filename.mdx
echo title:" " >> $filename.mdx
echo description:" "  >> $filename.mdx
echo date: \"`date +"%Y-%m-%dT%TZ"`\" >> $filename.mdx
echo tags: [] >> $filename.mdx
echo --- >> $filename.mdx
open $filename.mdx
```
